# Movie Recommender System

A content-based movie recommendation system built using Python, Scikit-learn, and Streamlit.

## Features
- Recommends similar movies using cosine similarity
- Displays movie posters using TMDB API
- Interactive web interface built with Streamlit

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit
- Requests

## How It Works
The system uses content-based filtering and cosine similarity to recommend movies similar to the selected movie.

## Run Locally

pip install -r requirements.txt

streamlit run app.py
